# 📄 คู่มือรวบรวม API ทั้งหมด (apiall.md)

เอกสารฉบับนี้รวบรวม API ทุกเส้นในระบบ **Waste-to-Coins** เพื่อใช้เป็นข้อมูลอ้างอิงสำหรับนักพัฒนา โดยแบ่งตามหมวดหมู่และการเข้าถึง

---

## 🏥 ตรวจสอบระบบ (System)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `GET` | `/health` | Public | ตรวจสอบสถานะเซิร์ฟเวอร์ | ใช้เช็คว่า Backend ยังทำงานปกติอยู่หรือไม่ |

---

## 🔐 ระบบสมาชิก (Authentication)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `POST` | `/api/auth/register` | Public | ลงทะเบียนผู้ใช้ใหม่ | ส่งข้อมูล User ID, ชื่อ, อีเมล และรหัสผ่านเพื่อสร้างบัญชี |
| `POST` | `/api/auth/login` | Public | เข้าสู่ระบบ | ส่งอีเมลและรหัสผ่านเพื่อรับ **JWT Token** มาใช้ยืนยันตัวตน |

---

## ♻️ การจัดการขยะ (Waste)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `POST` | `/api/waste/submit` | `auth` | ส่งข้อมูลขยะ | ผู้ใช้ระบุประเภท น้ำหนัก และรูปถ่าย เพื่อขอสะสมเหรียญ |
| `GET` | `/api/waste/my-submissions` | `auth` | ดูรายการขยะที่ตนเองส่ง | ตรวจสอบสถานะว่าขยะที่ส่งไป (Pending/Approved) เป็นอย่างไร |
| `GET` | `/api/waste/pending` | `officer` | ดูรายการขยะที่รอการอนุมัติ | **(เจ้าหน้าที่)** ใช้เพื่อดูว่ามีใครส่งขยะมาใหม่บ้าง |
| `POST` | `/api/waste/approve` | `officer` | อนุมัติการส่งขยะ | **(เจ้าหน้าที่)** กดยืนยันน้ำหนักและสั่ง Mint เหรียญให้ผู้ใช้ |

---

## 💰 กระเป๋าเงินและธุรกรรม (Wallet & Transactions)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `GET` | `/api/wallet/balance` | `auth` | ตรวจสอบยอดเหรียญ | เช็คยอดคงเหลือของเหรียญ WST ในกระเป๋า |
| `GET` | `/api/wallet/info` | `auth` | ดูข้อมูลที่อยู่กระเป๋าเงิน | ดูเลข Wallet Address ของตนเอง |
| `POST` | `/api/wallet/transfer` | `auth` | โอนเหรียญ | โอนเหรียญ WST ให้เพื่อนโดยระบุ Address ปลายทาง |
| `GET` | `/api/wallet/export` | `auth` | ส่งออก Private Key | **(คำเตือน: อันตราย)** ใช้เพื่อเอา Key ไปใส่ใน Metamask |
| `GET` | `/api/transactions/history` | `auth` | ดูประวัติธุรกรรมล่าสุด | ตรวจสอบประวัติการรับโอนเหรียญ (20 รายการล่าสุด) |

## 👮 สำหรับเจ้าหน้าที่ (Officer Operations)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `POST` | `/api/officer/add-coins` | `officer` | เติมเหรียญให้ผู้ใช้ | **(เจ้าหน้าที่)** ใช้เติมเหรียญให้ผู้ใช้เป็นกรณีพิเศษ |
| `GET` | `/api/officer/transactions` | `officer` | ดูธุรกรรมทั้งหมดในระบบ | ตรวจสอบความเคลื่อนไหวการ Mint เหรียญทั้งหมด |
| `GET` | `/api/officer/rewards/report` | `officer` | รายงานสต็อกและการแลก | ดูว่าของรางวัลอะไรเหลือเท่าไหร่ ใครแลกอะไรไปบ้าง |
| `GET` | `/api/users/list` | `officer` | รายชื่อผู้ใช้ทั้งหมด | ตรวจสอบข้อมูลสมาชิกในระบบ (เฉพาะบทบาท User) |

## 👤 จัดการโปรไฟล์ (Users & App Dashboard)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `GET` | `/api/users/profile` | `auth` | ดูโปรไฟล์และสถิติ | ดูข้อมูลส่วนตัวและจำนวนเหรียญรวมที่เคยได้รับ |
| `PUT` | `/api/users/profile` | `auth` | แก้ไขข้อมูลส่วนตัว | อัปเดตรูปโปรไฟล์ ชื่อ หรือเบอร์โทรศัพท์ |
| `POST` | `/api/users/change-password` | `auth` | เปลี่ยนรหัสผ่าน (จาก Profile) | เปลี่ยนรหัสผ่านโดยใช้รหัสเก่ามายืนยัน |
| `GET` | `/api/app/dashboard` | `auth` | ข้อมูลหน้าแรก (Aggregate) | ดึงข้อมูลชุดเดียว (Profile + ยอดเงิน + ธุรกรรม) มาโชว์ในหน้าแรก |
| `POST` | `/api/app/verify-identity` | `auth` | ยืนยันรหัสผ่าน | ใช้ตรวจสอบรหัสผ่านก่อนเข้าสู่หน้าสำคัญๆ |
| `POST` | `/api/app/change-password` | `auth` | เปลี่ยนรหัสผ่าน (จาก App) | ใช้ในขั้นตอนตั้งค่ารหัสผ่านใหม่ |

## 🎁 ของรางวัล (Rewards)
| Method | Path | Auth | จุดประสงค์ | วิธีใช้ |
| :--- | :--- | :--- | :--- | :--- |
| `GET` | `/api/rewards/list` | `auth` | รายการของรางวัล | ดูของรางวัลที่มีในสต็อกและราคาเหรียญ |
| `POST` | `/api/rewards/redeem` | `auth` | แลกของรางวัล | ใช้เหรียญ WST เพื่อแลกรับสิ่งของ |
| `GET` | `/api/rewards/history` | `auth` | ประวัติการแลกรางวัล | ตรวจสอบสถานะการรับของรางวัลที่แลกไปแล้ว |
| `POST` | `/api/rewards/add` | `officer` | เพิ่มของรางวัล | **(เจ้าหน้าที่)** ลงทะเบียนของรางวัลชิ้นใหม่ |
| `PUT` | `/api/rewards/update/:id` | `officer` | แก้ไขของรางวัล | **(เจ้าหน้าที่)** อัปเดตราคาหรือสต็อกของรางวัลเดิม |
| `DELETE` | `/api/rewards/delete/:id` | `officer` | ลบของรางวัล | **(เจ้าหน้าที่)** นำของรางวัลออกจากระบบ |
